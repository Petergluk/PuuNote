- Имеет **Plugin Registry** (пока без плагинов)

- Может запускать **фоновые jobs** (без интеграции)

|-----------|-----------|

| State | Zustand (разделён на 2 store) |
| Database | IndexedDB (Dexie) + snapshots table |
| Notifications | sonner (toast) |
| Tests | Native test runner (судя по .test.ts) |

## 3.2 Проблемы производительности 🟡

### 3.2.1 documentApi создаёт новые массивы

**Файл:** `src/domain/documentTree.ts:6-9`

```typescript
updateContent: (nodes, id, content) => {
  const targetNode = nodes.find((n) => n.id === id);
  if (targetNode && targetNode.content === content) return nodes;
  return nodes.map((n) => (n.id === id ? { ...n, content } : n));
}
```

**Проблема:** `nodes.map()` создаёт новый массив на каждый вызов, даже если изменения нет.

**Влияние:** При частом обновлении content (typing) — много аллокаций.

**Решение:** Оптимизировать check:
```typescript
updateContent: (nodes, id, content) => {
  const targetNode = nodes.find((n) => n.id === id);
  if (!targetNode || targetNode.content === content) return nodes;
  
  // Create new array only if content differs
  return nodes.map((n) => n.id === id ? { ...n, content } : n);
}
```

### 3.2.2 documentApi.moveNode создаёт copy всех nodes

**Файл:** `src/domain/documentTree.ts:118-119`

```typescript
const copy = nodes.map((n) => ({ ...n })); // ← shallow copy всех узлов
```

**Проблема:** Для перемещения одного узла копируются все.

**Влияние:** O(n) аллокаций на каждую операцию.

**Решение:** Копировать только затронутые узлы.

### 3.2.3 getDepthFirstNodes вызывается multiple times

**Файл:** `src/domain/contextExtraction.ts:74-78`

```typescript
const allDfNodes = getDepthFirstNodes(nodes).filter(df => descSet.has(df.id));
const targetDepthInfo = getDepthFirstNodes(nodes).find(n => n.id === targetNodeId);
const baseDepth = targetDepthInfo?.depth || 0;
```

**Проблема:** `getDepthFirstNodes(nodes)` вызывается дважды.

**Влияние:** O(2n) вместо O(n).

**Решение:**
```typescript
const allDfNodes = getDepthFirstNodes(nodes);
const targetDepthInfo = allDfNodes.find(n => n.id === targetNodeId);
```
